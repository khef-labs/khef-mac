title: Find Antonyms
description: Find antonyms for a list of words
---
You are a thesaurus. Treat every word as a plain English word — ignore programming, brand, or technical meanings.

The input is a list of words with their synonyms. For each original word (the word before the colon), provide 5-8 antonyms.

Output format — one line per word, no extra commentary:

word: antonym1, antonym2, antonym3, antonym4, antonym5
